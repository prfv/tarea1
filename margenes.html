Márgenes CSS
Las propiedades de margen CSS se usan para crear espacio alrededor de elementos, fuera de cualquier borde definido.

Con CSS, tienes control total sobre los márgenes. Hay propiedades para establecer el margen para cada lado de un elemento (arriba, derecha, abajo e izquierda).
Margen: lados individuales
CSS tiene propiedades para especificar el margen para cada lado de un elemento:

margen superior
margen derecho
margen inferior
margen izquierdo
Todas las propiedades de margen pueden tener los siguientes valores:

auto: el navegador calcula el margen
longitud: especifica un margen en px, pt, cm, etc.
%: especifica un margen en% del ancho del elemento contenedor
heredar: especifica que el margen debe heredarse del elemento padre
Consejo: se permiten valores negativos.
Ejemplo
Establezca diferentes márgenes para los cuatro lados de un elemento < p >:
< !DOCTYPE html >
< html >
< head >
< style >
div {
  border: 1px solid black;
  margin-top: 100px;
  margin-bottom: 100px;
  margin-right: 150px;
  margin-left: 80px;
  background-color: lightblue;
}
< / style >
< / head >
< body >

< h2 >Estamos usando propiedades de margenes individuales < / h2 >

< div >Esta division contiene margen: superior de 100px(pixeles),inferior de 100px, derecho de 150 e izquierdo de 80px < / div >

< / body >
< / html >


Margen - Propiedad abreviada
Para acortar el código, es posible especificar todas las propiedades de margen en una propiedad.

La propiedad de margen es una propiedad abreviada de las siguientes propiedades de margen individuales:
*margin-top
*margin-right
*margin-bottom
*margin-left
Pues así es como funciona:

Si la propiedad margen tiene cuatro valores:

margen: 25px 50px 75px 100px;
margen superior es 25px
margen derecho es 50px
margen inferior es 75px
margen izquierdo es 100px
Ejemplo
< !DOCTYPE html >
< html >
< head >
< style >
div {
  border: 1px solid black;
  margin: 25px 50px 75px 100px;
  background-color: lightblue;
}
< / style >
< / head >
< body >

< h2 >La propiedad acortada para poner los cuatro margenes  < / h2 >

< div >Esta division tendra un margen:superior de 25px, derecho de 50px inferior de 75, izquierdo de 100px< / div >

< / body >
< / html >

El valor automático
Puede establecer la propiedad de margen en automático para centrar horizontalmente el elemento dentro de su contenedor.

El elemento ocupará entonces el ancho especificado y el espacio restante se dividirá en partes iguales entre los márgenes izquierdo y derecho.
Ejemplo:
< !DOCTYPE html >
< html >
< head >
< style >
div {
  width:300px;
  margin: auto;
  border: 1px solid red;
}
< / style >
< / head >
< body >

< h2 >Usando margin:auto< / h2 >
< p >

< div >
Esta division automaticamente se centrara horizontalmente ya que contiene margin:auto
< / div >

< / body >
< / html >
