Las propiedades de fondo CSS se utilizan para definir los efectos de fondo para los elementos.

Propiedades de fondo de CSS:
*background-color
*background-image
*background-repeat
*background-attachment
*background-position

CSS background-color
La propiedad especifica el color de fondo de un elemento.
< !DOCTYPE html >
< html >
< head >
< style >
body {
  background-color: lightblue;
}
< / style >
< / head >
< body >

< h1 >Hola mundo< / h1 >

< p >Esta pagina tiene un azul claro de color de fondo< / p >

< / body >
< / html >
