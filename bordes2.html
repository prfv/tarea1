CSS bordes

CSS posicionamiento

Las propiedades compuestas border-top, border-right, border-bottom y border-left permiten establecer de forma independiente los cuatro
bordes(arriba, derecha, abajo e izquierda, respectivamente) de un elemento, definiendo su color, su grosor y su estilo. Se pueden
 escribir las tres  características de cada borde en cualquier orden. Se puede definir uno,dos, tres o cuatro bordes.
HTML
< p > hola < /p >
CSS
p {
  border-top: black 5px dashed;
  border-right: green 10px double;
  border-bottom: blue 10px dotted;
  border-left: red 5px solid;
}
